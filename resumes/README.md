# 履歷存放目錄

在此目錄下放置你的履歷，格式為 Markdown。

## 檔案命名

`{你的名字}.md`，例如：`allen.md`

## 格式範例

```markdown
---
name: 你的名字
email: your@email.com
preferred_roles: AI Engineer, Fullstack Developer
preferred_locations: Remote, US, Europe
---

# 技能

- Python
- JavaScript
- React
- AI/ML
- TypeScript
- AWS

# 經驗

5+ 年開發經驗...
```

## 權限

- 此目錄僅有你可以讀寫
- 請勿在此放敏感個資密碼
